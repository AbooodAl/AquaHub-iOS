# Aqua Hub - iOS Mobile Application

A native iOS application built with Swift & SwiftUI for water service management in Arar, Saudi Arabia.

## Features
- Order bottled water
- Schedule tank cleaning
- Secure payments (Apple Pay & Cash on Delivery)
- Real-time order tracking
- Admin dashboard

## Technologies
- **Frontend:** Swift, SwiftUI
- **Backend:** PostgreSQL
- **Architecture:** MVVM
- **Methodology:** Agile

## Project Structure
- `/Documentation` - Project reports and documentation
- `/Source_Code` - iOS application source code
- `/Database` - SQL schema and scripts
- `/Assets` - Design mockups and screenshots
